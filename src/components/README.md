# Proyecto: Viajando con Gonzalo

Aplicación web desarrollada con React, TypeScript y Vite.

## Stack del proyecto

- React + TypeScript → Construcción de la interfaz
- Vite → Servidor de desarrollo y compilación del proyecto
- Tailwind CSS → Estilos y diseño
- Supabase → Base de datos y servicios backend
